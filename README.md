### if文

| 特性 | 説明 |
|------|------|
| 役割 | 条件に基づいて処理を分岐させる |
| 基本構文 | `if (条件) { // 条件が真の場合の処理 } else { // 条件が偽の場合の処理 }` |
| 使用例 | ユーザーの年齢による表示内容の変更、入力値のバリデーションなど |

### for文

| 特性 | 説明 |
|------|------|
| 役割 | 指定された回数だけ処理を繰り返す |
| 基本構文 | `for (初期化式; 条件式; 更新式) { // 繰り返し処理 }` |
| 使用例 | 配列やリストの要素を順に処理する、一定回数繰り返す処理など |
